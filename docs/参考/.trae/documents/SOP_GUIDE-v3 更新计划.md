# SOP 文档审查指南更新计划（v2.12.0 → v3.0.0）

> **制定时间**: 2026-03-01  
> **目标**: 将审查指南从 Skill-first 架构更新为 Spec-first 架构

---

## 一、核心变更

### 1.1 架构理念转变

| 项目 | v2.12.0（旧） | v3.0.0（新） |
|------|--------------|-------------|
| **核心理念** | Skill-first（Skill 是唯一执行单元） | Spec-first（规范是核心，Skill 是实现方式） |
| **SSOT 定义** | Skill 矩阵、状态字典、命令字典 | 工程宪章、规范分层、契约模板 |
| **审查起点** | `sop/02_skill_matrix/` | `sop/AGENT_SOP.md` + `sop/01_constitution/` |
| **执行单元** | Skill | 规范驱动的 Skill |

### 1.2 目录结构变更

**旧结构（v2.12.0）**：
```
sop/
├── 02_skill_matrix/
├── 03_workflow/
├── 04_reference/
├── 05_constraints/
└── skills/
```

**新结构（v3.0.0）**：
```
sop/
├── AGENT_SOP.md          # 唯一入口
├── 01_constitution/       # P0 级：工程宪章
├── 02_specifications/     # P1-P2 级：系统规范
├── 03_workflow/           # 5 阶段工作流
├── 04_skills/             # Skill 定义（规范驱动）
├── 05_constraints/        # P0-P3 约束
├── 06_templates/          # 模板体系
└── 07_reference/          # 参考资料
```
