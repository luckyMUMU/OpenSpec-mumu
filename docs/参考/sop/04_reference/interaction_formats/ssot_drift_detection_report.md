## 修复建议

### 状态引用漂移

**修复流程**：
1. 确认引用的状态是否为预期状态
2. 如为预期状态 → 在 `state_dictionary.md` 中新增定义
3. 如为非预期状态 → 修改文档引用为已定义状态

### 命令引用漂移

**修复流程**：
1. 确认引用的命令是否为预期命令
2. 如为预期命令 → 在 `command_dictionary.md` 中新增定义
3. 如为非预期命令 → 修改文档引用为已定义命令

### 约束引用漂移

**修复流程**：
1. 确认引用的禁止项是否为预期禁止项
2. 如为预期禁止项 → 在 `constraint_matrix.md` 中新增定义
3. 如为非预期禁止项 → 修改文档引用为已定义禁止项
