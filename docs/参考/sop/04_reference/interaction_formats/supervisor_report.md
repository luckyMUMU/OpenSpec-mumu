# Supervisor 交互格式

**使用角色**: Supervisor  
**用途**: 进度报告、熔断警告、决策请求

---

## 进度通知格式

### 标准进度更新

```markdown

📈 **完成进度**: [已完成任务]/[总任务] ([百分比]%)
⏱️ **预计剩余**: [时间估算]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**当前阶段**: [详细描述]

**已完成**:
- [任务1]
- [任务2]

**进行中**:
- [任务3]

**待开始**:
- [任务4]
```

## 决策请求格式

### 标准决策请求

```markdown
🤔 **决策请求**
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
❓ **需要决策**: [决策事项]
📊 **当前状况**: [背景信息]
🔍 **影响分析**: [影响范围]
💭 **推荐方案**: [建议方向]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**决策背景**:
[详细背景]

**可选方案**:

**方案A**: [描述]
- ✅ **优点**: [优点]
- ❌ **缺点**: [缺点]

**方案B**: [描述]
- ✅ **优点**: [优点]
- ❌ **缺点**: [缺点]

**推荐**: 方案[A/B]

**请用户选择**:
- [ ] 采用方案A
- [ ] 采用方案B
- [ ] 提供其他方案
- [ ] 暂停任务
```

## 使用指南

### 何时使用

- **进度通知**: 每个任务完成后、达到里程碑时
- **熔断警告**: 触发三错即停机制时
- **决策请求**: 需要用户做出选择时
- **风险预警**: 发现潜在风险时
- **状态汇总**: 每日/每周定期报告

### 格式选择

| 场景 | 推荐格式 |
|------|----------|
| 常规进度更新 | 标准进度更新 |
| 重要里程碑 | 里程碑达成 |
| 初次失败 | 标准熔断警告 |
| 三次失败 | 严重熔断警告 |
| 常规决策 | 标准决策请求 |
| 紧急决策 | 紧急决策请求 |
| 潜在风险 | 风险预警 |
| 每日报告 | 日报格式 |
| 每周报告 | 周报格式 |
