# Supervisor报告格式

**使用**: Supervisor

---

## 进度更新

```markdown
## 进度更新

### 状态
- 任务: [名称]
- 阶段: [阶段]
- 进度: [N]/[M] ([%]%)

### 完成
- [任务1]
- [任务2]

### 进行中
- [任务3]

### 风险
- 🟡 [警告]
- 🔴 [严重]

### 下一步
@[角色]: [任务]
```

---

## 熔断警告

```markdown
## 熔断警告 ⚠️

### 触发
- 原因: [原因]
- 次数: [N]/3

### 问题
- Strike 1: [问题] → [处理]
- Strike 2: [问题] → [处理]
- Strike 3: [问题] → **熔断**

### 建议
- A: [方案]
- B: [方案]

**等待用户决策**
```

---

## 决策请求

```markdown
## 决策请求

### 事项
[描述]

### 选项
| 方案 | 优点 | 缺点 |
|------|------|------|
| A | [优点] | [缺点] |
| B | [优点] | [缺点] |

### 推荐
方案[A/B] - [理由]

**请用户选择**
- [ ] 方案A
- [ ] 方案B
- [ ] 其他
```

---

## 风险预警

```markdown
## 风险预警 ⚡

### 风险
- 类型: [类型]
- 等级: [高/中/低]

### 影响
- [后果1]
- [后果2]

### 措施
- [措施1]
- [措施2]
```
