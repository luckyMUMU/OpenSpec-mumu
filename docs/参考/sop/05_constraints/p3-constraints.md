# P3 级约束

> **约束强度**: 实现细节，IDE 实时提示

---

## 概述

P3 级约束定义实现细节，包括编码规范、测试规范、文档规范，由自动化工具实时验证。

---

## 编码规范

### P3-CODE-001: 缩进规范

| 属性 | 值 |
|------|-----|
| **描述** | 使用 2 空格缩进（或项目配置） |
| **验证方法** | IDE 插件（Prettier、EditorConfig） |
| **违反处理** | 自动格式化 |
| **例外流程** | 无例外 |

### P3-CODE-002: 行长度限制

| 属性 | 值 |
|------|-----|
| **描述** | 每行不超过 100 字符 |
| **验证方法** | IDE 插件 |
| **违反处理** | 自动换行 |
| **例外流程** | URL 可例外 |

### P3-CODE-003: 空行规范

| 属性 | 值 |
|------|-----|
| **描述** | 函数之间空一行，类之间空两行 |
| **验证方法** | IDE 插件 |
| **违反处理** | 自动格式化 |
| **例外流程** | 无例外 |

### P3-CODE-004: 引号规范

| 属性 | 值 |
|------|-----|
| **描述** | 优先使用单引号 |
| **验证方法** | IDE 插件 |
| **违反处理** | 自动格式化 |
| **例外流程** | 字符串包含单引号时可使用双引号 |

## 注释规范

### P3-COMMENT-001: 注释语言

| 属性 | 值 |
|------|-----|
| **描述** | 注释使用中文（或项目配置） |
| **验证方法** | 代码审查 |
| **违反处理** | 提示，建议修改 |
| **例外流程** | 引用外部文档时可使用英文 |

### P3-COMMENT-002: TODO 格式

| 属性 | 值 |
|------|-----|
| **描述** | TODO 注释应包含负责人和日期 |
| **验证方法** | 代码风格检查 |
| **违反处理** | 提示，建议补充 |
| **例外流程** | 无例外 |

**格式**: `// TODO(负责人): 描述 - YYYY-MM-DD`
